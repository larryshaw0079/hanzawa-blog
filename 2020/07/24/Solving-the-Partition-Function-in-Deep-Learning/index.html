Introduction

很多情况下我们需要计算下式： \[ p(\mathbf x;\theta)=\frac{\tilde{p}(\mathbf x;\theta)}{Z(\theta)} \] 其中\(\tilde{p}(\mathbf x;\theta)\)是Unnormalized的概率，\(Z(\theta)\)为正则化项，在离散的情况下\(Z(\theta)=\sum_x\tilde{p}(x)\)，在连续的情况下\(Z(\theta)=\int\tilde{p}(x)\mathrm d x\)。在很多情况下\(Z(\theta)\)是不可解的。

一个比较常见的场景是Softmax，比如对于NLP任务，神经网络的最后一层是Softmax，这时\(\tilde{p}(\mathbf x;\theta)=e^{s(\mathbf x|\mathbf c_t,\theta)}\)可以理解为一个“打分函数”：在给定Context的情况下，下一个词\(\mathbf x\)的概率分布。而\(Z(\theta)=\sum_{x\in X} e^{s(x|\mathbf x_t,\theta)}\)，最后在使用极大似然估计最大化整个训练集的似然，不过困难在于词表太大，\(Z(\theta)\)计算效率很低。

下面我们介绍处理这种情况的几个方法。

Noise Contrastive Estimation

Noise Contrastive Estimation（NCE）的算法细节及详细推导见原文。

我们将公式(1)两边取对数得到\(\log p(\mathbf x)=\log\tilde{p}(\mathbf x)-\log Z(\theta)\)，我们令\(c=-\log Z(\theta)\)，同时为了便于区分数据分布和模型分布加上了“model”标识： \[ \log p_\text{model}(\mathbf x)=\log\tilde{p}_\text{model}(\mathbf x;\theta)+c \] 也就是说，我们不仅是在优化\(\theta\)，同时也加入了一个新的参数\(c\)。需要注意的是上述式子无法直接用极大似然估计来优化，因为最大化上式没有上界，会导致结果无限大。

一句话概括NCE的主要思想就是通过将估计\(p(\mathbf x)\)这个无监督问题转化为了一个有监督问题，而在优化这个有监督问题的时候等价于优化原来的问题。

具体的来说，首先我们引入一个新的分布\(p_\text{noise}(\mathbf x)\)，称作是“噪声分布”。理论上\(p_\text{noise}(\mathbf x)\)可以采用任何易于采样的分布。之后我们引入一个新的随机变量\(y\)，表示样本是否来自模型分布： \[ p_\text{joint}(y=1)=\frac{1}{2}\\ p_\text{joint}(\mathbf x|y=1)=p_\text{model}(\mathbf x)\\ p_\text{joint}(\mathbf x|y=0)=p_\text{noise}(\mathbf x)\\ p_\text{joint}(y=1|\mathbf x)=\frac{p_\text{model}(\mathbf x)}{p_\text{model}(\mathbf x)+p_\text{noise}(\mathbf x)} \] 接着我们分别从真实数据和噪声分布中采样样本\(\mathbf x\)，两者数量一样（因为先验\(p_\text{joint}(y=1)=\frac{1}{2}\)），同时对应的\(y\)我们也是知道的。于是采用极大似然估计，最大化训练样本在我们的模型\(p_\text{joint}\)的似然： \[ \theta^*,c^*=\mathop{\arg\max}_{\theta,c}\mathbb E_{\mathbf x,y\sim p_\text{train}}\log p_\text{joint}(y|\mathbf x) \] 这里\(p_\text{joint}(y|\mathbf x)\)相当于Logistic回归： \[ \begin{align} p_\text{joint}(y=1|\mathbf x)&=\frac{p_\text{model}(\mathbf x)}{p_\text{model}(\mathbf x)+p_\text{noise}(\mathbf x)}\\ &=\frac{1}{1+\frac{p_\text{noise}(\mathbf x)}{p_\text{model}(\mathbf x)}}\\ &=\frac{1}{1+\exp\left(\log\frac{p_\text{noise}(\mathbf x)}{p_\text{model}(\mathbf x)}\right)}\\ &=\sigma\left(-\log\frac{p_\text{noise}(\mathbf x)}{p_\text{model}(\mathbf x)}\right)\\ &=\sigma(\log p_\text{model}(\mathbf x)-\log p_\text{noise}(\mathbf x)) \end{align} \] 如果样本\(\mathbf x\)是真实数据中采样而来，那么优化会试图最大化\(\sigma(\log p_\text{model}(\mathbf x)-\log p_\text{noise}(\mathbf x))\)，即最大化\(\log p_\text{model}(\mathbf x)-\log p_\text{noise}(\mathbf x)=\log\tilde{p}_\text{model}(\mathbf x;\theta)+c-p_\text{noise}(\mathbf x)\)，
